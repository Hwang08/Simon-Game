Press Any Key to Start

Rules

  1. Start, SIMON will give a first signal
  2. Repeat the signal by pressing the same button color.
  3. Remeber the preceding signals and SIMON will add one.
  4. Repeat these two signals by pressing the same button color.
  5. Remeber the preceding signals two signals and SIMON add another one.
  6. Continue in this manner as long as you can repeat each se-quence of signals correctly.
